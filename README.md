Arduino Bluetooth Robotic Car

Project Overview

This project is a Bluetooth-controlled robotic car developed using Arduino Uno, HC-05 Bluetooth module, and L293D Motor Shield. The car can be controlled wirelessly through a smartphone application and is capable of moving forward, backward, left, right, and stopping based on user commands.

Components Used

- Arduino Uno
- HC-05 Bluetooth Module
- L293D Driver
- DC Motors
- Chassis
- Wheels
- Battery Pack
- Switch
- Jumper Wires

Features

- Wireless Bluetooth control
- Forward and backward movement
- Left and right turning
- Real-time command execution
- Compact robotic vehicle design

Learning Outcomes

Through this project, I gained hands-on experience in:

- Arduino programming
- Bluetooth communication using HC-05
- Motor control using L293D Motor Shield
- Hardware assembly and integration
- Troubleshooting and debugging embedded systems

Project Note

This project was implemented by studying an open-source Bluetooth robotic car project and modifying the code during testing, debugging, and development. The project helped me understand the practical implementation of embedded systems and robotics concepts.
